# Java CLI Calculator

A simple command-line calculator built using Java fundamentals.  
This project was created to practice basic Java concepts such as methods, loops, switch-case, and user input handling.

---

## Features
- Addition
- Subtraction
- Multiplication
- Division (with divide-by-zero handling)
- Power (integer exponent)
- Modulus
- Square root
- Menu-driven program
- Runs in the terminal

---

## Technologies Used
- Java
- Scanner class for input

---

## How to Run

1. Make sure Java is installed on your system.
2. Open a terminal or command prompt.
3. Navigate to the project directory.
4. Compile the program
   javac src/Main.java
